This directory is used to make an EMACS which works with VTS (MIT
Virtual Terminals, used in the SUPDUP protocol).

CONFIG.MID contains a terminal type code for VTS==26. which should
match .TTSUP in the monitor.

In TECTRM.MID, for VTS terminals VCIVI/VCIVC (which are used for fs
InvMode$) have been implemented using the new .VTBOW function of VTSOP
(which translates to %TDBOW/%TDRST for Supdup).

A successful compilation relies on a fresh NMIDAS compiled with a
fresh TNXDFU generated from a fresh MONSYM.UNV, containing the VTS
definitions.

See EMACS.CTL for compilation instructions.
